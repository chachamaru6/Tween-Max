# Tween-Max
# 概要
オリジナルアプリ第二弾、gsapのTweenMaxを用いて動くホームページを実装しました。
# 制作背景
できるだけ多くの言語やライブラリに触れたいと思い制作しました。

動くホームページは見る人に大きなインパクトを与えると思います。これからよりレベルの高いアニメーション作成に取り組む予定です。
